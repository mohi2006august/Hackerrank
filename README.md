## ⚙️ Compiling & Running

Each folder contains a standalone `.cpp` file. Compile and run any solution with:

```bash
g++ -std=c++17 -O2 -o solution path/to/solution.cpp
./solution < input.txt
```

- **Compiler:** g++ / any C++17-compatible compiler
